# Todo List Application
Modern task manager with priority-based planning, smart filters, editing flow, and a responsive MUI interface.

Live demo: [https://oleksandrprotasov.github.io/Todo_list_application/](https://oleksandrprotasov.github.io/Todo_list_application/)

## What is inside
- Create, edit, complete, and delete tasks.
- Priority levels for each task (`High`, `Medium`, `Low`).
- Task search by title.
- Filters by status and priority.
- Sorting (`Newest first`, `Oldest first`, `By priority`).
- Local storage persistence between sessions.
- Responsive modern UI with Material UI components.

## Stack
- React 18
- Material UI (MUI)
- Emotion (`@emotion/react`, `@emotion/styled`)
- React Scripts (CRA)

## Local setup
Install dependencies:

`npm install`

Start development server:

`npm start`

## Scripts
- `npm start` - run local development server
- `npm run build` - create production build
- `npm test` - run tests
- `npm run deploy` - build and publish to GitHub Pages

## Notes
- The app is configured for GitHub Pages deployment.
- Project data is saved in browser local storage.
